# Switch

## Example

```javascript
<Switch checked={checked} onChange={this.handleChange}>
  Checkbox
</Switch>
```

## props

|   属性   |          类型          | 默认值 |     描述     |
| :------: | :--------------------: | :----: | :----------: |
| checked  |       `boolean`        |   -    |   是否选中   |
| onChange | `Function(checked, e)` |   -    | 变化时的回调 |
