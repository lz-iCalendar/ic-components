# Button

## Example

```javascript
<Button type="primary">Button</Button>
```

## props

| 属性  |   类型    |  默认值   |              描述               |
| :---: | :-------: | :-------: | :-----------------------------: |
| type  | `string`  | `primary` | 按钮类型，可选 `primary` `text` |
| block | `boolean` |  `false`  |      宽度是否为父元素宽度       |
