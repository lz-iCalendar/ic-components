# FieldsBoardData

字段面板数据组件。基于 FieldsBoard 组件（对表单字段的管理）

## Example

```javascript
<FieldsBoardData formManagementResid={111111} />
```

## props

|        属性         |   类型   | 默认值 |           描述            |
| :-----------------: | :------: | :----: | :-----------------------: |
| formManagementResid | `string` |   -    | formManagement 表的 resid |
