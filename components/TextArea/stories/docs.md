# TextArea

## Example

```javascript
<TextArea placeholder="请输入内容" value={value} onChange={this.handleChange} />
```

## props

|   属性   |     类型      | 默认值 |          描述          |
| :------: | :-----------: | :----: | :--------------------: |
|  value   |   `string`    |   -    |           值           |
| onChange | `Function(e)` |   -    | 输入框内容变化时的回调 |
